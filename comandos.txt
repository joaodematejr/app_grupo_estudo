Run instructions for iOS:
    • cd appGrupoEstudo && react-native run-ios
    - or -
    • Open appGrupoEstudo/ios/appGrupoEstudo.xcodeproj in Xcode or run "xed -b ios"
    • Hit the Run button

  Run instructions for Android:
    • Have an Android emulator running (quickest way to get started), or a device connected.
    • cd appGrupoEstudo && react-native run-android
